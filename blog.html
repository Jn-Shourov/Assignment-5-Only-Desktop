Question-1: What are the different ways to select an element in the DOM?

There are 5 different ways to select an element in the DOM. 1. document.getElementsByTagName(Select one or more elements of HTML by its/their tag or element name), 2. document.getElementById(Select one HTML element by its unique id), 3. document.getElementsByClass(Select one or more elements of HTML by its/their class name), 4. document.querySelector(Select one element of HTML by its CSS selector), 5. document.querySelectorAll(Select one or more elements of HTML by its/their CSS selector).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The differences between them are :

1. 'innerHTML' gives us each and every character written in that perticular element or tag, even other HTML contents, just not the comments.
2. 'innerText' gives us only the text or string that is shown on the webpage, written in that element ignoring other things.
3. 'textContent' gives us only the text that is written in that element ignoring other things like 'innerText' but whether its shown or hidden in webpage by CSS properties, it will give us every text in between that element. So, CSS does not applied to it.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating an element and showing how to add and remove that element, using JavaScript:

1. const newElement = document.createElement('Tag Name Of HTML Element');
2. document.body.appendChild(newElement);
3. document.body.removeChild(newElement);
or,
newElement.remove();

This is how to create (.createElement('Element Tag')), add (.appendChild()) and remove (.removeChild() / ) an element using JavaScript.
